# Стани анімації кульок

## Послідовність станів

| # | Стан | Опис | Тривалість |
|---|------|------|------------|
| 1 | **INIT** | Порожня сцена, 3 чистих кадри | ~50ms |
| 2 | **ПАДІННЯ** | Кульки падають зверху, колізії, обертання | 2-3 сек |
| 3 | **СПОКІЙ** | Кульки лежать на дні, не рухаються | ∞ |
| 4 | **ВІТЕР** | Кульки літають вгору, турбулентність | toggle |
| 5 | **WIN** | Велика фіолетова куля з анімацією появи | 1.5 сек |
| 6 | **RESET** | Видалити все, створити нові кульки | миттєво |

---

## WIN послідовність

Кнопка "WIN послідовність" запускає повний цикл:

```
0ms      → Вмикається вітер (кульки трясуться)
2000ms   → Вітер вимкнено, з'являється WIN куля
         → Малі кульки втягуються в центр і зникають
3500ms   → Відкривається попап
         → Сцена скидається до стану СПОКІЙ
```

---

## Тестові кнопки

| Кнопка | Дія |
|--------|-----|
| **Тест трясіння** | Toggle вітру |
| **Показати WIN** | Тільки WIN куля (без послідовності) |
| **WIN послідовність** | Повний цикл: трясіння → WIN → попап → reset |
| **Скинути сцену** | Примусовий reset до початкового стану |
| **Показати попап 1/2** | Відкрити попап з призом |
| **Закрити попапи** | Закрити всі відкриті попапи |
